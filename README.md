# Sawrail
Sawrail is a basic game engine / rendering engine for XY oscilloscopes.

## Development
1. Run `git clone https://github.com/charnam/sawrail.git`
2. Place the downloaded files into an HTTP server. The easiest way to do this is to run `python3 -m http.server`
3. Visit `http://localhost:8000`, or whatever domain the code is being hosted on.
